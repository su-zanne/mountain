HOW TO START THIS PROJECT: 
Run the main method in the ClubStats class to start the program.
From there you should follow the instructions on screen in order
to add climbers, add mountains, and view statistics. You can exit
the program from the main menu.

PURPOSE OF PROJECT:
The program will enable mountain climbers to keep track of the mountains
that they have climbed. It will also provide statistics so that the
climbers can compare their achievements.
